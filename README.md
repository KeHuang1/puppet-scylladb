# Puppet ScyllaDB role

This role installs and configures ScyllaDB cluster.

## Requirements

Platform requirements are listed in the metadata file.

## Documentation

[ScyllaDB documentation](http://docs.scylladb.com/)

## Simple setup

```puppet
class{ 'scylla':
  dc    =>  'ScyllaIL',
}
```
